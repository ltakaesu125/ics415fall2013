Included Technologies

This webpage is built mainly using the Twitter Bootstrap CSS and Javascript package, enabling the website to scale easily for mobile devices as well as up to 1080p desktop monitors without the need to build multiple implementations for each viewport case. Instead, Bootstrap adaptive elements will automatically resize, collapse, and reposition themselves in order to still be readable in any resolution case.

In order to add more interaction with the user, jQuery is used to animate certain portions of the website, such as the slideshow on the home page. In addition, jQuery is combined with PHP and MySQL in order to parse and login a user. Access to the website is mostly unrestricted, except when attempting to view order history. PHP is also used to store and modify cookies on the client-side in order to determine if an indivual is logged in.
